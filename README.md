# GESTOR PRESUPESTO

## Descripción

Este proyecto es una aplicación de control de gastos que te permite llevar un registro detallado de tus gastos y gestionar tu presupuesto de manera efectiva. Con esta herramienta, podrás tener un mejor control de tus finanzas personales y tomar decisiones informadas sobre tus gastos.

## Instrucciones de uso

Para acceder a la página, simplemente abra el archivo index.html en su navegador web
